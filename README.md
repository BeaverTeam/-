## 如何调试

### 前端

1. 确保 Node 和 npm 的版本比较新（参考：Node v7.6.0，npm v4.4.4）；
2. 输入指令 ```npm install -g ionic cordova``` 安装 Ionic 2（可能需要 sudo）；
3. 进入 /app 文件夹 ```cd app```；
4. 安装依赖 ```npm install```；
5. 运行指令 ```ionic serve```，调试程序将运行在 8100 端口；
